คู่มือการติดตั้งและใช้งานระบบ (Setup Guide)
สิ่งที่ต้องมีในเครื่องก่อนเริ่ม (Prerequisites)
ก่อนจะรันโปรเจกต์นี้ได้ เครื่องคอมพิวเตอร์หรือเซิร์ฟเวอร์ปลายทางจะต้องติดตั้งโปรแกรมพื้นฐานเหล่านี้ไว้ครับ:
Node.js (แนะนำเวอร์ชัน 18 ขึ้นไป หรือ 24 ตามที่โปรเจกต์ใช้)
Python (เวอร์ชัน 3.10 ขึ้นไป สำหรับรันระบบ AI)
FFmpeg (จำเป็นมากสำหรับการถอดเสียงวิดีโอ)
Mac: ติดตั้งผ่านคำสั่ง brew install ffmpeg
Windows: ต้องโหลดไฟล์มาติดตั้งและตั้งค่า Environment Variable
 ขั้นตอนการเตรียมโปรเจกต์ (Installation)
เมื่อได้ไฟล์โปรเจกต์มาแล้ว ให้เปิด Terminal (หรือ Command Prompt) ชี้ไปยังโฟลเดอร์โปรเจกต์ แล้วทำตามนี้ครับ
ติดตั้งไลบรารีของฝั่งเว็บ (Next.js)
	npm install
ติดตั้งไลบรารีของฝั่ง AI (Python)
	pip3 install -U openai-whisper

การตั้งค่าระบบ (Environment Variables)
จุดนี้สำคัญที่สุด ระบบจะพังถ้าลืมตั้งค่าครับ ให้สร้างไฟล์ชื่อ .env ไว้ที่โฟลเดอร์นอกสุดของโปรเจกต์ แล้วใส่ข้อมูลเหล่านี้ลงไป
# 1. URL สำหรับเชื่อมต่อฐานข้อมูล (เปลี่ยน user, password และชื่อ db ตามจริง) DATABASE_URL="postgresql://user:password@localhost:5432/upskill_db" 
# 2. URL ของเว็บเรา (ถ้าทำบนเครื่องตัวเองใช้ localhost ถ้าขึ้นเว็บจริงใช้โดเมนจริง) NEXTAUTH_URL="http://localhost:3000" 
# 3. รหัสลับสำหรับเข้ารหัสระบบล็อกอิน (พิมพ์ตัวอักษรหรือตัวเลขสุ่มๆ ยาวๆ ได้เลย) NEXTAUTH_SECRET="my_super_secret_key_123456789"

การเตรียมฐานข้อมูล (Database Setup)
ก่อนเปิดเว็บ ต้องสร้างตารางในฐานข้อมูลให้เรียบร้อยก่อนด้วยเครื่องมือ Prisma ครับ รัน 2 คำสั่งนี้ตามลำดับ
สร้าง Client สำหรับคุยกับฐานข้อมูล
	npx prisma generate
สร้างตารางในฐานข้อมูล (Push Schema)
npx prisma db push
เปิดใช้งานระบบ (Run the Application)
เมื่อทำทุกอย่างครบแล้ว ก็สามารถเปิดเซิร์ฟเวอร์รันหน้าเว็บได้เลยครับ
	npm run dev

จากนั้นเปิดเบราว์เซอร์แล้วเข้าไปที่ http://localhost:3000
