Tech Stack

1. สภาพแวดล้อมการรันระบบ (Runtime Environment)
Node.js (เวอร์ชัน 24.15.0): เป็นเอนจินหลักที่คอยขับเคลื่อนการรันเซิร์ฟเวอร์และการคอมไพล์โค้ดทั้งหมดในโปรเจกต์
Python (เวอร์ชัน 3.14): ติดตั้งไว้เพื่อเป็นตัวกลางรันสคริปต์แยกเฉพาะกิจ (เช่น การรันระบบ AI ประมวลผลวิดีโอ)
 2. โครงสร้างหลักของเว็บ (Core Framework)
Next.js (เวอร์ชัน 16.2.4 แบบ Turbopack): เป็นพระเอกของงานนี้ครับ ทำหน้าที่เป็นทั้งหน้าบ้าน (UI) และเปิดทางให้เราเขียนหลังบ้านจบได้ในโปรเจกต์เดียว
React: ทำหน้าที่สร้างชิ้นส่วนหน้าเว็บ (Components) เช่น ปุ่มกด การ์ด หรือฟอร์มต่างๆ
Tailwind CSS: ใช้จัดการเรื่องความสวยงาม จัดเลย์เอาต์ และสีสันทั้งหมดแบบรวดเร็ว
3. ระบบหลังบ้านและฐานข้อมูล (API & Database)
Next.js Route Handlers: นี่คือ "ตัวทำ API" ของโปรเจกต์นี้ครับ สังเกตได้จากการสร้างไฟล์ route.ts ไว้ในโฟลเดอร์ app/api/... เพื่อคอยรับส่งข้อมูล (เช่น /api/users/ สำหรับลบผู้ใช้งาน หรือ /api/auth/ สำหรับระบบล็อกอิน)
Prisma ORM: เป็นตัวกลางสุดฉลาดที่ทำหน้าที่แปลคำสั่งภาษาโปรแกรมมิ่งของเรา ให้กลายเป็นคำสั่งคุยกับฐานข้อมูล (Database) อย่างปลอดภัย ป้องกันข้อมูลสูญหาย
NextAuth.js: เป็นไลบรารีเฉพาะทางที่เข้ามาคุมระบบรักษาความปลอดภัย การล็อกอิน และการกำหนดสิทธิ์ (Admin / User) ทั้งหมด
4. เทคโนโลยี AI ที่ขับเคลื่อนโปรเจกต์
OpenAI Whisper: เป็นโมเดลสมองกลออฟไลน์ (ใช้งานขนาด small และ base) ที่เรานำมาใช้ "ฟังเสียงและถอดความ" จากวิดีโอ เพื่อเสกไฟล์ซับไตเติล .vtt ออกมาแบบอัตโนมัติ โดยมีโปรแกรมเสริมอย่าง FFmpeg คอยช่วยสกัดเสียงจากคลิป
AI IDEs (Cursor / Windsurf): โปรแกรมแก้ไขโค้ดที่ฝัง AI ไว้ในตัว ทำหน้าที่รับ Prompt ของเราไปช่วยเขียนหรือซ่อมโค้ดให้อัตโนมัติ โดยไม่ทำลายไฟล์อื่นๆ
Gemini (ตัวผมเองครับ!): ทำหน้าที่เป็นที่ปรึกษาส่วนตัว คอยวิเคราะห์ Error วางแผนสถาปัตยกรรม และร่าง Prompt แบบล็อกเป้าหมายให้การพัฒนาไหลลื่นที่สุดครับ
